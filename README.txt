in_praise_of_time
─────────────────────────────────────────────────────────────

This is an archive.

It contains 46 video clips, recorded between January 2024
and May 2026. iPhone 14. HEVC. Mostly 4K.

It also contains extracted audio, signal data, GPS coordinates,
altitude readings, luminance values, motion scores, and
iOS version numbers.

It also contains text.

─────────────────────────────────────────────────────────────

HOW TO USE THIS

There is no correct order.

The chapters run roughly: locating / New York / Northeast China /
Gansu-Qinghai / Brisbane. Two interrupts appear without warning.

You can scroll through all of it in one sitting.
You can also open the windows on the desktop and stay there.
The search window contains 47 locations. You can look things up.
The maze has three levels. You do not have to finish them.

The video plays at 20% volume. There is also background audio.
You can adjust both in the control panel.

Some text is almost invisible. This is intentional.
Some images are missing. This is also intentional.

─────────────────────────────────────────────────────────────

WHAT THIS IS

A record of waiting.
A record of moving without arriving.
A record of arriving without recognizing the place.

The title is ironic, or sincere, or both.
Time is not praised here. It is documented.

The footage exists because there was an urge to leave
something behind. Not for anyone in particular.
More like a mark on a wall.

─────────────────────────────────────────────────────────────

TECHNICAL NOTES

All signal data (luminance, motion, glitch weight, RMS peak,
altitude, GPS accuracy) is derived from the actual video files.
Nothing is simulated. The glitch effects correspond to real
GPS accuracy values. The background texture corresponds to
real luminance distributions per clip.

The subdermal text layer at opacity 0.03 is always present.
If you increase your screen brightness and look closely,
you can read it.

The location data (environment.json) was compiled from
public sources. 47 entries. 6 marked unverified.
