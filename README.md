# IN_PRAISE_OF_TIME

An interactive browser-based video and signal archive.

Open `index.html` locally, or serve the directory with any static web server so the browser can load `signal_data/`, `footage_report/`, and the video assets.

Large video and audio assets are tracked with Git LFS.
